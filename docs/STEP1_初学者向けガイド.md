# 📖 Step1: 初学者向けガイド
*所要時間: 約20分 | 対象: プログラミング初心者〜中級者*

## 🎯 このアプリは何？

**「No.1 Gift Search App（ギフト検索アプリ）」** は、冠婚葬祭のギフト選びを簡単にするWebアプリです。

### 💡 解決する問題
- 結婚祝い、出産祝い、母の日、父の日などのギフト選びが面倒
- 価格帯やレビューで絞り込みたい
- 適切な商品を素早く見つけたい

### ✨ アプリの特徴
- **2070件の厳選商品**から検索
- **用途別フィルタ**（結婚祝い・出産祝い・母の日・父の日など）
- **価格帯・レビュー順ソート**で希望にぴったりの商品を発見
- **高速検索**（1秒以内で結果表示）

---

## 🏗️ アプリの全体構成

```
🌐 ユーザーのブラウザ
    ↕️ (検索リクエスト・商品表示)
⚡ フロントエンド (Next.js)
    ↕️ (API呼び出し)
🔧 バックエンド (FastAPI)  
    ↕️ (検索クエリ)
🔍 検索エンジン (Meilisearch)
    ↕️ (データ参照)
💾 商品データベース (JSON)
```

### 🎭 各部分の役割

| 部分 | 技術 | 役割 | 例えるなら |
|------|------|------|-----------|
| **フロントエンド** | Next.js + React | ユーザーが見る画面 | お店の店頭・ショーケース |
| **バックエンド** | FastAPI + Python | データを処理する | お店の店員・レジ係 |
| **検索エンジン** | Meilisearch | 商品を素早く探す | お店の在庫管理システム |
| **データ** | JSON | 商品情報を保存 | お店の倉庫・商品カタログ |

## 🔄 アプリの動作フロー

### 👤 **ユーザーの操作**
1. ブラウザでアプリにアクセス
2. 「タオル」と検索キーワードを入力
3. 「母の日」のフィルタを選択
4. 「価格安い順」で並び替え
5. 気に入った商品をクリック

### ⚡ **システムの処理**
```
1. フロントエンド: ユーザー入力を受け取り
2. バックエンド: リクエストを受け取り
   ⚙️  検索条件を整理・バリデーション

3. 検索エンジン: 高速検索を実行
   🔍 2070件から条件に合う商品を0.1秒で発見

4. バックエンド: 結果をフロントエンドに返送
   ⚙️  "112件のタオル商品、価格安い順"

5. フロントエンド: 結果を美しく表示
   📱 商品カード形式で一覧表示
```

## 🚀 技術的な特徴

### ⚡ **高速性**
- **検索時間**: 0.1秒以内
- **ページ読み込み**: 1秒以内  
- **同時アクセス**: 100人まで対応

### 🎯 **検索精度**
- **日本語対応**: ひらがな・カタカナ・漢字
- **あいまい検索無効**: 意味のない結果を排除
- **関連度順**: 最適な商品を上位表示

### 📱 **使いやすさ**
- **レスポンシブデザイン**: スマホ・PC対応
- **直感的UI**: 迷わず操作できる
- **高速表示**: ストレスなく閲覧
