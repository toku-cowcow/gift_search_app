# 📚 初学者向け学習ガイド - No.1 Gift Search App

*プログラミング初心者でも20分×3ステップで理解できる、実践的なWebアプリ学習教材*

## 🎯 このガイドの特徴

✅ **段階的学習**: 3つのステップで無理なく理解  
✅ **実践重視**: 実際に動くアプリで学習  
✅ **現代技術**: 現場で使われる最新技術スタック  
✅ **初心者対応**: プログラミング経験が浅くても理解できる

## 📖 学習ステップ

### 📖 **Step 1: 初学者向けガイド** 
📁 [`STEP1_初学者向けガイド.md`](./STEP1_初学者向けガイド.md)

## 🎓 学習後のレベル

### ✅ **習得できるスキル**
- **フロントエンド**: React/Next.jsでのモダンUI構築
- **バックエンド**: FastAPIでのRESTful API開発
- **インフラ**: Docker Composeでのマルチサービス管理
- **検索システム**: 高速全文検索エンジンの運用
- **システム設計**: レイヤードアーキテクチャの理解

### 🌟 **応用できる分野**
- **ECサイト**: 商品検索・フィルタリング機能
- **ブログ・メディア**: 記事検索・カテゴリ分類
- **企業サイト**: FAQ検索・サイト内検索
- **マッチングサービス**: ユーザー・コンテンツ検索
- **データ分析**: ログ検索・可視化ツール

## 🚀 学習の進め方（推奨）

### 👤 **初心者の方**
```
1. Step1を読む (20分) → 全体像把握
2. 休憩 (10分)
3. Step2を読む (20分) → 技術理解  
4. 休憩 (10分)
5. Step3を実践 (20分) → 動かして体験
```

### 💪 **ある程度経験がある方**  
```
1. Step1をざっと読む (5分) → 概要確認
2. Step3で実際に動かす (15分) → 動作確認
3. Step2で技術詳細を学ぶ (20分) → 理解を深める
4. Step3で応用・カスタマイズ (20分) → スキルアップ
```

### 🏃 **時間がない方**
```
1. Step1の「アプリの動作フロー」だけ読む (5分)
2. Step3の「Step1: アプリを起動」だけ実行 (10分)
3. 実際にアプリを触ってみる (5分)
```
